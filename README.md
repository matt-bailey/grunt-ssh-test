# Gruntbase

> A simple Grunt base for web projects

## Clone the repo

```bash
git clone https://github.com/gpmd/gruntbase.git .
```

## Install the project dependencies

```bash
bower install
npm install
```

## Build options

**Development mode:**

```bash
grunt --env=dev
```

**Production mode:**

```bash
grunt --env=prod
```

**Styles reporting (SCSS lint / CSS stats) mode:**

```bash
grunt report
```
